# EGA-CMTNet
EGA-CMTNet: A Novel Multi-Domain Edge-Guided Adaptive Canny-Augmented Mamba-Transformer Network for Multi-Modality Image Fusion

## Installation 
- Python
- [vmamba](https://github.com/MzeroMiko/VMamba)
- Other dependencies listed in `requirements.txt`

## Pre-Processing 
Run
```
python dataprocessing.py
``` 

## Training
Run 
```
python train.py
``` 

## Testing
Run 
```
python test.py
```
